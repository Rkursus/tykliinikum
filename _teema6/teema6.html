1 Pakett ggplot2: Skaalade muutmine

Kui ggplot2-ga koostada joonis, siis muutujad seostatakse graafiliste elementide omadustega (asukoht x-teljel, asukoht y-teljel, vrvus, suurus jne). Vaikimisi kasutatakse kllalt mistlikke skaalasid, nt vrviskaalad on valitud selliselt, et kik vrvid oleks vrdse intensiivsusega. Mnikord on aga soov skaalasid muuta. Selleks saab kasutada kske scale_<graafilise elemendi omadus>_<skaala nimi>. Niteks scale_x_continuous(.) ksuga saab muuta x-telje vahemikku, scale_colour_grey(.) muudab vrviskaala mustvalgeks.

Kiki skaalade muutmise funktsioone on vimalik nha ggplot2 kodulehel http://ggplot2.tidyverse.org/reference/ jaotuse Scales all.

Skaleerimisfunktsiooni rakendamiseks tuleb see lisada joonisele:

p2 <- ggplot(mk, aes(per_capita_inc, unemployment_rate, colour = State)) + geom_point()
p2 + scale_x_continuous(trans = "log10", breaks = c(1/2, 1:4) * 10^4)

Eelneva ksuga lisati x-telje skaala muudatus: telg kuvatakse logaritmilisel skaalal, breaks mrab teljel esitatavad arvvrtused.

Iga erineva omaduse skaala muutmiseks saab joonisele juurde liita uue funktsiooni. Konkreetse funktsiooni parameetrid sltuvad omaduse tbist (nt punkti kuju ei saa logaritmida), aga kigil skaleerimisfunktsioonidel on kindlasti kolm argumenti:

  • name telgede puhul telje nimi; vrvide, kuju jm legendis antava info puhul vastava legendi pealkiri
  • breaks vektor vrtustega, mis mravad, millised vrtused joonisel ra markeeritakse (nt x-telje jaotis)
  • labels parameetriga breaks mratud punktidele vastavad sildid

x-ja y-telgi ja teisi skaalasid (mh nende nimesid) saab muuta vastava scale-ksuga, ent on veel vimalusi lisada telgedele/skaaladele nimed ja piiritleda vrtusvahemikku:

  • xlab(.), ylab(.) vastavate telgede pealkirjad
  • labs(.) saab mrata nii telgede, legendi elementide ka kui graafiku pealkirja (x = "...", y = "...", color = "...", , title = "...")
  • xlim(.), ylim(.) argumendiks kaks vrtust, mis mravad telgede vrtusvahemiku. NB! objektid, mille vrtused on vljaspool mratud vahemikku asendatakse NA vrtusega ja jetakse vlja kigilt kihtidelt.
  • lims(.) ldine skaalapiiride mramise ksk, lisaks x- ja y-telgedele saab piire mrata ka niteks vrviskaalale. Kui vrviskaala on seotud osariigi nimega, siis mramaks vrvi ainult kahele osariigile tuleks ksk kirja panna kujul: lims(color = c("Alaska", "Texas")).
  • coord_cartesian(.) argumentidega xlim, ylim saab mrata xy-teljestikus vrtusvahemikud, mis nhtavale jvad. See on vimalus joonise mingile piirkonnale suumida. Joonisest like tegemine.
  • ggtitle(.) joonise pealkiri.

Teljepiiride mramine ylim(.) vi xlim(.) ksuga vib mjutada joonisel esitatavaid arvutuslikke tulemusi. Niteks kui joonisel esitada keskvrtused koos usalduspiiridega ning mrata teljepiirid nii, et mni arvutuse aluseks olev vaatlus jb piiridest vlja, siis muutuvad ka usalduspiirid. Nide sellisest olukorrast on jrgmisel joonisel (siin on kasutuses on teine niteandmestik: rottide kehakaalu ja dieetide andmestik)

rotid <- nlme::BodyWeight  
rotid <- rotid[rotid$Diet != 1, ]  
p <- ggplot(rotid, aes(factor(Time), weight)) + geom_point(color = "gray") + 
  stat_summary(geom = "errorbar", fun.data = mean_se, fun.args = list(mult = 1.96)) +
  stat_summary(geom = "point", fun.y = mean)
p
p + ylim(390, 600)

1.1 Pidevate/arvuliste skaalade muutmine

Pidevate skaalade muutmise kskudel (nt scale_<?>_continuous, scale_<?>_gradient) on mned spetsiifilised argumendid:

  • trans skaala transformeerimise funktsiooni nimi, nt exp, log, log10, sqrt.
  • limits kahe-elemendiline vektor, mis annab skaala algus- ja lpp-punkti. Sarnalset xlim(.), ylim(.) kskudega tuleb selle argumendi kasutamisel olla thelepanelik, sest andmed, mis vastavast vahemikust vlja jvad asendatakse NA vrtustega. Selle toimingu mju sltub muudetavast skaalast: x-ja y-telgede piiride mramisel thendab see, et neid andmeridu ei kasutata joonise tegemisel (nt ka regressioonisirge arvutamisel). Vrviskaala korral mrab see objektid, millele vrvi ei mrata(vt alapunkt Vrviskaala muutmine).

Arumendi trans kasutamise nide on toodud praktimijuhendi alguses: x-telje skaala on sellel joonisel logaritmiline.

Argumendi limits mju vrrelduna graafikult like tegemisega (coord_cartesian) vaata jrgmistel joonistel regressioonsirgeid vrreldes.

p2 + geom_smooth(method = lm) + scale_x_continuous(limits = 1:2*10^4) +
  labs(title = "x-teljel kasutame ja neme\npunkte vahemikus 10000-20000")  
p2 + geom_smooth(method = lm) + coord_cartesian(xlim = 1:2*10^4) +
  labs(title = "kasutame kiki punkte, tulemust\nneme vahemikus 10000-20000")

1.2 Diskreetsete skaalade muutmine

Diskreetsetel skaaladel ttab argument limits teistmoodi: nimelt saab sellega ette anda konkreetsed vrtused, mida joonisel kujutatakse, lejnud vrtusi siis joonisel ei kujutata. Oluline on ka vrtuste etteandmise jrjekord:

b <- ggplot(data = mk, aes(State, unemployment_rate)) + geom_boxplot() 
b + scale_x_discrete(limits = c("Texas", "Maryland", "Alaska")) 
#Vrdle, mis mju on argumendil `breaks`:
# b +  scale_x_discrete(breaks = c("Texas", "Maryland", "Alaska"))

Kui joonise vrvid on mratud diskreetsel skaalal, siis limits argumendiga saab mrata millisete vrtusklasside elemendid vrvitakse ja millised jetakse vrvimata (vaikimisi jvad halliks):

b1 <- ggplot(data = mk, aes(bachelor, unemployment_rate, color = State)) + geom_point()
b1 + scale_color_discrete(limits = c("Texas", "Maryland", "Alaska")) 
# NA vrtuse vrvi saa ka muuta
b1 + scale_color_discrete(limits = c("Texas", "Maryland", "Alaska"), na.value = "white")

joonis <- ggplot(data = mk, aes(x = high_scl, y = bachelor)) + geom_point()

1.3 Vrviskaala vrvide muutmine

Vrvide valik joonisel on vga oluline. igesti valitud vrvidega on vimalik tuua selgemini vlja oma snumit, muuta joonist loetavamaks ja meeldivamaks. Asjakohane vrvus sltub tunnusest, mida soovitakse kujutada. ldiselt vib vrviskaalad jaotada kolmeks:

  • gradient pidevate tunnuste jaoks; kige viksem ja kige suurem vrtus vastavad mingitele vrvidele ning vahepealsed vrtused nende kahe vrvi segule
  • lahknev gradient pidevate tunnuste jaoks, kui pideval tunnusel on mingi selge nullpunkt (nt hutemperatuur, tsentreeritud skoor vms); kaks ekstreemset vrtust ja nullpunkt vastavad mingile puhtale vrvile, vahepealsed vrtused kahe vrvi segule.
  • kvalitatiivne diskreetsete tunnuste jaoks; iga vrtuse jaoks kasutatakse vimalikult erinevat vrvitooni. Samas on oluline silmas pidada, et heleduselt ja intensiivsuselt oleks kik vrvid vrdsed.

Gradientskaalat saab kontrollida ksuga scale_<?>_gradient(.) (ksimrgi asemel on tavaliselt fill vi colour). Saab kasutada kiki pideva tunnuse skaala muutmise argumente, ning lisaks on kaks argumenti: low ja high ekstreemsete vrtuste vrvi mramiseks gradiendil. Lahknevat gradienti saab kontrollida funktsiooniga scale_<?>_gradient2(.), millel on lisaks low ja high vrtustele argument mid millega saab ette vrvi nime, mis vastab nullpunktile, vaikimisi on see vrv valge. Skaala nullpunkti vrtuse saab ka ise ette anda. Lahknevat ja veel keerulisemaid gradiente saab mrata funktsiooniga scale_<?>_gradientn, mille argumendile colours saab ette anda vektori vrvidega, mille vahele siis uued vrvid sujuva leminekuga valitakse.

Nited kahe gradientskaala ksu kasutamisest:

p3 <- ggplot(data = mk, aes(per_capita_inc, unemployment_rate)) + 
  geom_point(aes(colour = bachelor ))
nimi = "% w/ higher \n education"
p3 + scale_colour_gradient(name = nimi, low = "yellow", high = "red")
p3 + scale_colour_gradient2(name = nimi, low = "blue", high = "red", midpoint = 30)

Diskreetsete/kvalitatiivsete vrviskaalade kontrollimiseks kasutatakse vaikimisi funktisooni scale_<?>_hue(.), mis valib HCL vrviskaalal2 parameetri h (hue vrvitoon) vrtused vimalikult erinevad, jttes vrvi tugevuse c (chroma) ja heleduse l(luminance, lightness) konstantseks. Nii saadakse vimalikult erinevad vrvid, mis samal ajal on hesuguse intensiivsusega.

p4 <- ggplot(mk, aes(per_capita_inc, unemployment_rate)) + geom_point(aes(colour = State ))
p4 
p4 + scale_colour_hue(c = 150) # tugevus suuremaks, vaikimisi 100
#p4 + scale_colour_hue(l = 20) # heledus madalamaks, vaikimisi 65
#p4 + scale_colour_hue(h = c(10, 190)) # kaks vrvitooni, skaala algus- ja lpptoon

Diskreetsete vrviskaalade jaoks on arendatud vrvipalett Colorbrewer3 (algselt arendatud maakaartide vrvimiseks). ggplot2-s pseb sellele paletile ligi funktsiooniga scale_<?>_brewer(.), millel on kaks argumenti:

  • type vimalikud vrtused on "seq", "div" ja "qual"
  • palette paleti number (vt http://www.colorbrewer2.org)
p4 + scale_colour_brewer(type = "div", palette = 2) 
p4 + scale_colour_brewer(type = "qual", palette = 2)

2 Joonise viimistlemine

Vaikimisi joonistab ggplot2 halli taustaga jooniseid et heledamad vrvid oleksid sama silmatorkavad kui tumedad. Kui on soov valge tausta jrele, siis seda saab tellida, lisades joonisele ksu + theme_bw(). Veelgi detailsemalt saab joonise kujundust ksuga + theme(.), mille argumentidest mned olulisemad on:

Argument Elemendi tp Selgitus
line line kik jooned
rect rect kik ristklikulised elemendid (taustad, raamid)
text text kik tekstid
axis.line line joonise teljed
axis.text.x text x-telje vrtused
axis.text.y text y-telje vrtused
axis.ticks line joonise jaotised
axis.title.x text x-telje pealkiri
axis.title.y text y-telje pealkiri
legend.background rect legendi taust
legend.key rect legendi vtme taust
legend.text text legendi tekst
legend.title text legendi pealkiri
panel.background rect graafiku taust
panel.border rect graafikut mbritsev raam
panel.grid.major line jaotise jooned
panel.grid.minor line jaotise jooned
plot.background rect kogu joonise taust
plot.title text joonise pealkiri
strip.background rect tahkude pealkirjade taust
strip.text.x text horisontaalsete tahkude pealkiri
strip.text.y text vertikaalsete tahkude pealkiri

Tielikum loetelu theme(.) vimalikest argumentidest on aadressil http://ggplot2.tidyverse.org/reference/#section-themes .

Nagu nha, on kolm peamist elemenditpi, mida saab muuta kskudega element_text(.), element_line(.) ja element_rect(.). Nendel kskudel on omakorda argumendid:

  • element_text(.) family, size, colour, angle, hjust, vjust kontrollivad teksti rifti, suurust, vrvi, kaldenurka ja positsiooni vaikimisi mratud koha suhtes. Niteks size=14 valib teksti punktisuuruseks 14pt.
  • element_line(.) colour, size, linetype;
  • element_rect(.) colour, size, linetype re muutmiseks, fill sisemuse vrvi muutmiseks

Kui mnda elementi ldse joonisel ei soovi nha, siis tuleb vastavale argumendile anda vrtus element_blank().
